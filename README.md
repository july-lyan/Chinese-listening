# 听听看 - 中文听力练习微信小程序

## 项目介绍

「听听看」是一款中文听力练习微信小程序，帮助用户提高中文听力水平。用户可以选择不同难度的练习，通过听音频选择正确的答案，完成练习后可以查看得分和答题回顾。

## 功能特点

- **难度选择**：提供初级、中级、高级三个难度级别
- **听力练习**：支持句子选择和图片选择两种题型
- **音频播放**：使用微信内置的语音合成功能播放音频
- **实时反馈**：答题后即时显示正确与否及详细解析
- **成绩统计**：计算得分并显示答题回顾
- **排行榜**：保存成绩并显示排行榜
- **数据持久化**：使用Supabase云数据库存储成绩，本地存储作为备用

## 技术栈

- 微信小程序原生框架
- Supabase（云数据库）
- 微信内置语音合成API

## 如何运行

1. 下载微信开发者工具
2. 克隆本项目到本地
3. 在微信开发者工具中导入项目
4. 点击「编译」按钮运行小程序

## 使用说明

1. **选择难度**：在首页选择适合自己的难度级别
2. **开始练习**：进入练习页面，点击「播放」按钮听音频
3. **选择答案**：根据听到的内容选择正确的答案
4. **查看反馈**：答题后会显示正确与否及详细解析
5. **完成练习**：完成所有题目后，会自动跳转到结果页面
6. **查看结果**：在结果页面查看得分、答题回顾和排行榜

## 注意事项

- 本项目使用Supabase云数据库存储成绩，需要确保网络连接正常
- 如果Supabase连接失败，会自动使用本地存储作为备用
- 音频播放使用微信内置的语音合成功能，需要在微信环境中运行

## 开发说明

- 问题数据位于 `data/questions.js` 文件中，可以根据需要添加或修改问题
- 样式使用了暖黄教育风，与Web版保持一致
- 响应式布局适配不同屏幕尺寸
